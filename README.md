# Robotics

## Projects
### Wall Follow
Robot follows a wall and turns appropriately around and in corners  
Uses ultrasonic sensors to determine distances to walls  

## Variables
### Motors
M1 = Right  
M2 = Left  

### Speed
SR = Right Motor speed  
SL = Left Motor speed  
Values from 0 - 255  

### Direction
DR = Right motor direction  
DL = Left motor direction  
Low value = forward  
High value = backward
